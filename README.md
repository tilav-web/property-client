# Amaar Properties — Web

Malayziya ko'chmas mulk bozorining mijozlar va admin uchun web-interfeysi. React 19 + Vite asosida qurilgan SPA: ko'p tilli (uz / ru / en / ms), AI orqali tabiiy tilda qidiruv, xarita orqali e'lonlarni ko'rish, real-time chat va admin panel.

## Texnologiyalar

- **Framework:** React 19 + Vite 7 + TypeScript
- **Stil:** Tailwind CSS v4 + Radix UI
- **State:** Zustand (mahalliy holat) + TanStack Query (server holati)
- **Form:** React Hook Form + Zod / Formik + Yup
- **Router:** React Router 7
- **i18n:** i18next (uz / ru / en / ms)
- **Xarita:** Google Maps + Leaflet
- **Real-time:** Socket.IO Client

## Loyihani ishga tushirish

```bash
# 1. Bog'liqliklarni o'rnatish
npm install

# 2. .env faylini sozlash (VITE_API_URL, VITE_GOOGLE_MAPS_KEY, va h.k.)

# 3. Development rejimda ishga tushirish
npm run dev

# 4. Production build
npm run build
npm run preview
```

## Foydali skriptlar

```bash
npm run lint     # ESLint tekshiruvi
npm run build    # TypeScript + Vite build
```
